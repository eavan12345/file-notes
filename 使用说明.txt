共有四个文件：

  1. file-notes.py     — 命令行小工具（Python 脚本）
  2. SKILL.md          — Claude Skill，在会话中运行 /file-notes 使用
  3. filenotes.bat      — 在地址栏输入filenotes查询文件夹功能
  4. 使用说明.txt       — 本文件

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  一、Python 工具用法
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

打开终端（PowerShell/cmd），运行：

  1. 查询路径记录
     python C:\Users\XIE\.claude\skills\file-notes\file-notes.py D:\某个文件夹

  2. 非交互式添加记录（推荐）
     python C:\Users\XIE\.claude\skills\file-notes\file-notes.py add D:\某个文件夹 --desc "这是XX项目的存档"

  3. 交互式添加记录（会弹问答）
     python C:\Users\XIE\.claude\skills\file-notes\file-notes.py add D:\某个文件夹

  4. 列出所有已记录的路径
     python C:\Users\XIE\.claude\skills\file-notes\file-notes.py list

  5. 导出为易读文本
     python C:\Users\XIE\.claude\skills\file-notes\file-notes.py export

  6. 查看帮助
     python C:\Users\XIE\.claude\skills\file-notes\file-notes.py

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  二、设置快捷命令（推荐，省得每次输长路径）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

在 PowerShell 中运行以下命令（仅当前会话有效）：

  function file-notes { python "$env:USERPROFILE\.claude\skills\file-notes\file-notes.py" @args }

之后就可以直接敲：

  file-notes D:\某个文件夹
  file-notes add D:\某个文件夹 --desc "备注"
  file-notes list
  file-notes export

想永久有效：把这行命令加到你的 PowerShell profile 里
（用 notepad $PROFILE 打开，粘贴进去保存）

  三、文件管理器地址栏查询（最方便）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. 把 filenotes.bat 复制到 C:\Windows\System32
     这样在文件管理器的地址栏输入 filenotes 就能直接运行

  2. 打开文件管理器，进入你想查的文件夹

  3. 在地址栏（路径框）里输入 filenotes 然后按回车

  4. 弹出一个 cmd 窗口，显示该文件夹的记录

  四、Claude Skill 用法（批量扫描）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

在 Claude Code 会话中直接输入：

  /file-notes D:\某个路径

Claude 会：
  1. 列出指定路径下的所有子文件夹
  2. 根据文件夹名称推理用途（不确定的会看里面文件辅助判断）
  3. 批量记录到 ~/.file-notes/data.json
  4. 展示汇总清单给你过目

也可以只查不扫：

  /file-notes           → 告诉 Claude 你想做什么（查/扫/导出）

建议：用电脑一段时间后，跑一次 /file-notes D:\ 来更新记录。

  四、数据文件
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

所有记录保存在：

  C:\Users\XIE\.file-notes\data.json

格式是 JSON，每个条目包含：
  - path        — 原始路径
  - status      — 状态（默认 unsure）
  - description — 用途描述
  - updated_at  — 记录日期
